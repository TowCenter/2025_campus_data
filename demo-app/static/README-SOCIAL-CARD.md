# Social Media Card Image

## How to Generate the Social Card PNG

The social card HTML file (`social-card.html`) is ready. To create the PNG image:

### Method 1: Browser Screenshot (Easiest)
1. Open `social-card.html` in your browser
2. Use your browser's developer tools:
   - Right-click on the card → Inspect
   - Select the `.social-card` element
   - Right-click → Capture node screenshot
   - Save as `social-card.png` in this folder

### Method 2: Manual Screenshot
1. Open `social-card.html` in your browser
2. Zoom to 100% (the card is exactly 1200x630px)
3. Take a screenshot of just the card
4. Crop to exactly 1200x630px
5. Save as `social-card.png` in this folder

### Method 3: Online Tool
1. Open `social-card.html` in your browser
2. Copy the HTML/CSS
3. Use https://htmlcsstoimage.com/ or similar tool
4. Set dimensions to 1200x630px
5. Download and save as `social-card.png`
